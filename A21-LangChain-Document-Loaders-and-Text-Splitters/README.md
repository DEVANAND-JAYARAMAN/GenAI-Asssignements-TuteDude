# Assignment 21: LangChain Document Loaders & Text Splitters

## Files Used

- notes.txt
- employees.csv
- DevAnand_Resume.pdf

## Libraries Used

- LangChain
- LangChain Community
- LangChain Text Splitters
- PyPDF
- BeautifulSoup4

## Tasks Performed

- Loaded text, CSV, PDF, directory, and web documents using LangChain document loaders.
- Explored document metadata and content.
- Split large documents using CharacterTextSplitter and RecursiveCharacterTextSplitter.
- Created a reusable function to load and split documents from local folders and web URLs.
- Compared different text splitting approaches and recorded observations.


This assignment provided hands-on experience with LangChain document loaders and text splitters. These techniques form the foundation for building Retrieval-Augmented Generation (RAG) systems and AI-powered document assistants.
