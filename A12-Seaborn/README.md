## How to Run

1. Install the required libraries

pip install pandas 
pip install matplotlib 
pip install seaborn

2. The CSV file should be inside the A12-Seaborn folder

3. run the program using this command:

python task.py

if you run this, the plot/image ill open. If you close that plot/image then another next plot will be opned.
