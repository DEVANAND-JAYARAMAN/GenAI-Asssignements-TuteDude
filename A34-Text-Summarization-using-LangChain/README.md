# Assignment 34 - Text Summarization using LangChain

This project demonstrates different text summarization techniques using LangChain and a Large Language Model (LLM)

Features

* Prompt-based summarization
* Stuff summarization chain
* Map-Reduce summarization chain
* Refine summarization chain
* Reusable summarization function

Technologies Used

* Python
* LangChain
* Groq LLM
* Python Dotenv


How to Run

1. Install the required packages:
   pip install -r requirements.txt

2. Create a .env file and add your Groq API key:
   GROQ_API_KEY=your_api_key

3. Run the program:
   python app.py
